# Spec 034: Animations & Transitions

## Priority: P2

## Description
Add smooth page transitions between Inertia pages and micro-animations on interactive elements.

## Acceptance Criteria
- [ ] Subtle fade or slide transition during page navigation
- [ ] No transition on redirects (login, etc.)
- [ ] Transitions do not block user interaction
- [ ] Smooth performance (no jank)
- [ ] Cards: hover effect (slight scale + shadow) with CSS transition
- [ ] Buttons: subtle hover effect
- [ ] Homepage sections: progressive reveal on scroll (intersection observer)
- [ ] Animations respect `prefers-reduced-motion`
